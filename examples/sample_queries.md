# Sample FinOps Queries

Once OpsYield MCP FinOps Server is connected to your AI agent (e.g., Claude), you can copy and paste the following queries to start exploring your cloud infrastructure costs and optimizations.

## 💰 Cost Summaries

**AWS**
> "Show me the cost summary for AWS over the last 30 days. Group the cost by service."

**GCP**
> "What is my total GCP spend for the current billing cycle? Are there any unexpected spikes?"

**Azure**
> "Can you generate a summary of my Azure costs for the last week? Break it down by Resource Group."

**Multi-Cloud**
> "Give me a high-level summary of my cloud costs across AWS, GCP, and Azure for this month."

## 🕵️ Waste & Anomaly Detection

> "Analyze my cloud environments for any idle resources or wasted spend."

> "Can you find any unattached disks or orphaned IPs in my AWS and GCP environments?"

> "Are there any cost anomalies or sudden spikes in my AWS usage over the last 14 days?"

## 📉 Rightsizing & Optimization

> "Based on current utilization, do you have any rightsizing recommendations for my EC2 or Compute Engine instances?"

> "Identify any underutilized compute resources in Azure that I could scale down to save money."

## 🏗️ Architecture & Planning

> "I want to deploy a new web service. Based on my current spending patterns across AWS and GCP, which cloud provider would be more cost-effective for hosting a Kubernetes cluster and a managed PostgreSQL database?"

> "If I delete all the idle resources you found, how much money will I save per month?"
